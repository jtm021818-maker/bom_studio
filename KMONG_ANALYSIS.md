# KMONG.COM - Comprehensive UI/UX Analysis
## Korean Freelance Marketplace Platform

**Analysis Date:** February 13, 2026  
**Platform:** https://kmong.com/  
**Purpose:** Design reference for AI video outsourcing platform redesign

---

## 1. HOMEPAGE/LANDING PAGE ANALYSIS

### 1.1 Header Structure
**Location:** Sticky header at top (76px height)  
**Components:**
- **Logo:** "kmong" text logo on left (clickable, links to home)
- **Navigation Menu:** Horizontal menu with main categories
  - 엔터프라이즈기업용 (Enterprise)
  - 전문가 등록 (Expert Registration)
  - 로그인 (Login)
  - 회원가입 (Sign Up)
- **Category Navigation:** Dropdown/horizontal menu showing:
  - 디자인 (Design)
  - IT·프로그래밍 (IT/Programming)
  - 영상·사진·음향 (Video/Photo/Audio)
  - 마케팅 (Marketing)
  - 전자책 (E-books)
  - 세무·법무·노무 (Tax/Legal/HR)
  - AI 서비스 (AI Services)
- **User Menu:** Right side with login/signup buttons
- **Design:** White background, sticky positioning, clean typography

### 1.2 Hero Section
**Structure:**
- **Headline:** Large Korean text "성공이 필요한 순간, 딱 맞는 전문가를 찾아보세요"
  - Translation: "When you need success, find the perfect expert"
- **Search Bar:** Prominent centered search input with placeholder "어떤 전문가가 필요하세요?"
  - Magnifying glass icon on right
  - Rounded corners, light gray background
- **Quick Category Tags:** Below search bar showing popular searches
  - 로고, 앱, 상세페이지, 라탑, 블로그, PPT, 전기요금, 포도산
- **Right Side Banner:** Promotional image/carousel showing AI services
  - Colorful gradient background (purple/blue)
  - Product showcase with "사주·타로·신점" (Tarot/Fortune telling)
  - Carousel indicator showing "2/5"

### 1.3 Category Navigation Section
**Layout:** Horizontal scrollable grid of category icons
**Categories Displayed:**
- 영상 (Video) - with "인기" (Popular) badge
- 뷰티 (Beauty) - with "인기" badge
- 디자인 (Design)
- IT프로그래밍 (IT/Programming)
- 영상·사진·음향 (Video/Photo/Audio)
- 마케팅 (Marketing)
- 전자책 (E-books)
- 세무·법무·노무 (Tax/Legal/HR)
- AI 서비스 (AI Services) - with "Best" badge
- 전자제기 (More categories)

**Design Pattern:**
- Each category has a colorful icon
- Icon + text label below
- Hover effects (likely color change or scale)
- Responsive: scrollable on mobile

### 1.4 Service Listing Sections
**Multiple Curated Sections:**

#### Section 1: "쇼핑몰 사장님이 많이 찾아요" (Popular with Shop Owners)
- **Layout:** 4-column grid (responsive)
- **Cards per row:** 4 on desktop, 2 on tablet, 1 on mobile
- **Card Structure:**
  - **Image:** Large thumbnail (16:9 aspect ratio)
  - **Title:** Service name (2-3 lines max)
  - **Rating:** Star rating + review count (e.g., "5.0 (300)")
  - **Price:** Starting price in Korean Won (e.g., "10,000원~")
  - **Seller Info:** Circular avatar + seller name
  - **Badges:** "인기" (Popular), "Best" badges on some cards

#### Section 2: "매장 운영할 때 많이 �아요" (Popular for Store Operations)
- Same card layout as above
- Different curated services

#### Section 3: "에너지창업자들이 많이 찾아요" (Popular with Entrepreneurs)
- Same card layout
- Different service category

**Card Design Details:**
- Rounded corners on images
- Hover effect: likely shadow or scale
- Clean typography with good contrast
- Price displayed prominently in green text
- Seller avatar is circular with border

### 1.5 Search Bar Behavior
- **Placement:** Hero section center
- **Style:** Rounded pill-shaped input
- **Placeholder:** "어떤 전문가가 필요하세요?" (What expert do you need?)
- **Icon:** Magnifying glass on right side
- **Quick tags:** Popular searches below for quick access

### 1.6 Footer Structure
**Multi-column layout:**
- **Column 1:** Company info, logo
- **Column 2:** Service categories links
- **Column 3:** Help/Support links
- **Column 4:** Legal/Policy links
- **Bottom:** Copyright, social links

## 2. CATEGORY PAGES ANALYSIS

**Filter Options:**

### 2.5 Pagination
**Style:** Not visible in standard view (likely infinite scroll or "Load More" button)

## 3. SERVICE DETAIL PAGE ANALYSIS

### 3.1 Header Section
**Layout:**
- **Breadcrumb:** Home > Category > Service name
- **Title:** Large heading with service name
- **Rating:** Star rating + review count prominently displayed
- **CTA Button:** "주문하기" (Order) button in purple/primary color

### 3.2 Gallery/Portfolio Display
**Structure:**
- **Main Image:** Large featured image (16:9 or similar)
- **Thumbnail Gallery:** Horizontal scrollable thumbnails below
- **Carousel Controls:** Previous/Next arrows
- **Image Count:** Shows current image number (e.g., "1/5")

### 3.3 Pricing Section
**Display:**
- **Base Price:** Starting price prominently shown
- **Price Tiers:** Multiple package options (if available)
- **Pricing Table:** Shows different service levels with features
- **CTA:** "주문하기" (Order) button for each tier

### 3.4 Freelancer/Seller Info Sidebar
**Location:** Right side or below main content  
**Information:**
- **Avatar:** Circular profile picture
- **Name:** Seller/freelancer name
- **Rating:** Overall rating and review count
- **Completion Rate:** Percentage of completed projects
- **Response Time:** Average response time
- **Member Since:** Join date
- **Portfolio Link:** Link to full portfolio
- **Contact Button:** "메시지 보내기" (Send message)

### 3.5 Reviews Section
**Structure:**
- **Review Count:** Total number of reviews
- **Average Rating:** Star rating breakdown
- **Review List:** Individual reviews with:
  - Reviewer name/avatar
  - Rating (stars)
  - Review text
  - Date posted
  - Helpful count (thumbs up)

### 3.6 Service Description
**Content:**
- Detailed service description
- What's included
- Delivery time
- Revision policy
- FAQ section

### 3.7 CTA Buttons
**Primary Actions:**
- "주문하기" (Order) - Main CTA in purple
- "메시지 보내기" (Send message) - Secondary CTA
- "찜하기" (Wishlist/Save) - Icon button
- "공유하기" (Share) - Icon button
